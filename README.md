# Social Data Science

Git repo for the summer school course, Introduction to Social Data Science, at the Faculty for Social Sciences at University of Copenhagen (Summer 2020), see webpage [here](https://abjer.github.io/isds2020).

Previous versions of the course material be seen here:  [2016](https://github.com/sebastianbarfort/sds_summer), [2017](https://github.com/abjer/sds2017), [2018](https://github.com/abjer/sds), [2019](https://github.com/abjer/sds2019). Note that in 2016 the summer school was run mainly by Sebastian Barfort teaching in R.
